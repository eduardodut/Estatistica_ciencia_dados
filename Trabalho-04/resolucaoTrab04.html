1 Enunciado

Objetivos: Analisar a distribuio da amostra onde o fenmeno Tempo agora diz respeito aos valores de 1 fator e 3 nveis Fazer os testes para anlise da varincia do fenmeno.

Considere o caso em que o tempo do usurio a varivel independente (calculada) e representando o tempo que o usurio passou em uma determinada conferncia virtual, quando fez uso de um dos Meet virtuais, se usando Zoom, Hangout ou Skype. A hiptese saber se existe diferena significativa entre os trs Meet. O arquivo segue os seguintes princpios para a realizao deste trabalho: - independncia dos dados, quem usou um meet no usou o outro. - A varivel tempo mais prxima de uma log normalidade, porque a medida que o usurio usa um sistema, ele se torna mais especialista e o tempo, no eixo X tende a diminuir com o tempo; ou ainda tem poucas atividades que levam muito tempo e muitas que levam pouco tempo, afetadas pela experincia do usurio.

Pede-se: Fazer os testes para anlise da varincia do fenmeno, realizando as trs tcnicas dadas a seguir e considerando que:

  1. a tcnica Shapiro-wilk permite testar a normalidade, para uma amostra pequena
  2. a tcnica Kolmogorov-Smirnov permite testar a lognormalidade da amostra.
  3. a tcnica Levene
  4. Visualize os dados usando Boxplot, histograma e qqplot.

Seno houver normalidade da amostra, ento transforme os dados em uma log normal, depois, verifique como ficaram os dados, e repita os testes dos passos.

Obs: In a test statistic: the result expresses in a single number how much my data differ from my null hypothesis. So it indicates to what extent the observed scores deviate from a normal distribution. Now, if my null hypothesis is true, then this deviation percentage should probably be quite small. That is, a small deviation has a high probability value or p-value. Reversely, a huge deviation percentage is very unlikely and suggests that my reaction times dont follow a normal distribution in the entire population. So a large deviation has a low p-value. As a rule of thumb, we reject the null hypothesis if p < 0.05. So if p < 0.05, we dont believe that our variable follows a normal distribution in our population.

gera_histograma <- function(dados, bins=9){

  n            <- length(dados$Tempo)   
  nome         <- deparse(substitute(dados))
  mediaAmostra <- mean(dados$Tempo) 

  sd<-sqrt(var(dados$Tempo)*(n-1)/n) 
  
  histograma <- ggplot(dados,aes(Tempo))
  histograma <- histograma + geom_histogram(bins = bins, 
                                            aes(y=..density.., fill=..count..))
  histograma <- histograma + labs( x="",y="Frequncia", 
                                   title=paste("Tempo utilizado por usurio na plataforma",
                                               nome))
  histograma <- histograma + scale_fill_gradient("Amostra por caixa", 
                                                 low="#DCDCDC", 
                                                 high="#7C7C7C",)
  histograma <- histograma + stat_function(fun=dnorm, 
                                           color="red", 
                                           args=list(mean=mediaAmostra, 
                                                     sd=sd))
  
  ylimHist= layer_scales(histograma)$x$range$range
  
  diagCaixa <- ggplot(dados, aes(y=Tempo))
  diagCaixa <- diagCaixa + geom_boxplot()
  diagCaixa <- diagCaixa + theme(axis.title.y=element_blank(),
                                 axis.text.y=element_blank(),
                                 axis.ticks.y=element_blank())
  diagCaixa <- diagCaixa + labs(y=paste("Tempo de uso do",nome))
  diagCaixa <- diagCaixa + coord_flip(ylim = ylimHist)
  
  plot_grid(histograma, diagCaixa,  
                     ncol = 1, rel_heights = c(2, 1),
                     align = 'v', axis = "rlbt")  
}

gera_ksplot <- function(dados, distribuicao){
  
  
  media    <- mean(dados$Tempo)
  sd       <- sd(dados$Tempo)
  nome     <- deparse(substitute(dados))
  nomeDist <- deparse(substitute(distribuicao))
  group    <- c(rep(nome, length(dados$Tempo)), 
             rep("Dist Normal", length(distribuicao)))
  dat      <- data.frame(KSD = c(dados$Tempo,distribuicao), group = group)

  cdf1 <- ecdf(dados$Tempo) 
  cdf2 <- ecdf(distribuicao) 

  minMax <- seq(min(dados$Tempo, distribuicao), 
                max(dados$Tempo, distribuicao), 
                length.out=length(dados$Tempo)) 
  x0 <- minMax[which(abs(cdf1(minMax) - cdf2(minMax)) == 
                       max(abs(cdf1(minMax) - cdf2(minMax))) )] 
  y0 <- cdf1(x0) 
  y1 <- cdf2(x0) 
  ggplot(dat, aes(x = KSD, group = group, color = group))+
    stat_ecdf(size=1) +
      theme(legend.position ="top") +
      xlab("Amostra") +
      ylab("ECDF") +
      #geom_line(size=1) +
      geom_segment(aes(x = x0[1], y = y0[1], xend = x0[1], yend = y1[1]),
          linetype = "dashed", color = "red") +
      geom_point(aes(x = x0[1] , y= y0[1]), color="red", size=2) +
      geom_point(aes(x = x0[1] , y= y1[1]), color="red", size=2) +
      ggtitle(paste("K-S Test: Plataforma",nome,"/",nomeDist)) +
      theme(legend.title=element_blank())
}

3.3 Testes estatsticos

Chegou-se a concluses distintas quanto normalidade utilizando o mtodo grfico e o clculo da assimetria e curtose.

necessrio portanto aplicar testes estatsticos de normalidade para a obteno de resultados mais conclusivos.

3.3.1 Teste de Shapiro-Wilk:

O teste de Shapiro-Wilk apresenta a estatstica W e o valor P para representar a significncia estatstica do teste. A hiptese nula :

H0: A amostra foi retirada de uma populao que segue uma distribuio normal.

A estatstica W do teste, varia de entre 0 e 1, quanto mais alto for W mais a amostra se aproxima de uma distribuio normal.

O teste tambm apresenta o valor de significncia estatstica valor p para a amostra em questo.

Se o valor p para uma dada amostra for menor que um nvel de significncia designado pode-se rejeitar a hiptese nula e afirmar que a amostra no segue uma distribuio normal. Valores comuns para comparao de testes de hipteses so: 0.1, 0.05, 0.01, a depender do que se est estudando e o nvel de rigor requerido.

Abaixo a amostra Zoom testada para normalidade seguindo o mtodo de Shapiro-Wilk:

testeZoom <- shapiro.test(Zoom$Tempo)
testeZoom
## 
##  Shapiro-Wilk normality test
## 
## data:  Zoom$Tempo
## W = 0.84372, p-value = 0.004191

A partir do teste aplicado nas amostras que utilizaram Zoom pode-se afirmar que:

A um nvel de significncia de 0.1, 0.05 ou 0.01 a hiptese nula pode ser rejeitada chegando-se a concluso que a amostra no vem de uma populao que segue uma distribuio normal.

O resultado do teste confirma o que foi visto atravs do desvio acentuado da assimetria e curtose da amostra e contraria a anlise grfica realizada.

O teste aplicado novamente removendo o valor extremo:

testeZoom <- shapiro.test(Zoom$Tempo[1:19])
testeZoom
## 
##  Shapiro-Wilk normality test
## 
## data:  Zoom$Tempo[1:19]
## W = 0.98052, p-value = 0.9482

O novo teste aplicado sem o valor extremo possui um valor p maior do que o maior valor normalmente utilizado de 0.1, portanto a distribuio segue uma distribuio normal se o valor extremo for excludo.

3.3.2 Teste de Kolmogorov-Smirnov

O teste de Kolmogorov pode ser utilizado para comparar duas amostras ou para comparar uma amostra com uma distribuio padro.

O teste de Kolmogorov apresenta a estatca D: Mxima diferena absoluta entre duas funes de distribuies cumulativas e possui um valor P para representar a significncia estatstica do teste. O teste de Kolmogorov possui as seguintes hipteses nulas:

Comparao entre duas amostras:
H0: As duas amostras foram retiradas de uma populao com a mesma distribuio.

Comparao entre uma amostra e uma distribuio de referncia:
H0: A amostra foi retirada de uma populao que segue a distribuio de referncia.

Aplica-se ento o teste para comparar a amostra Zoom a uma distribuio normal de mdia e desvio padro iguais aos da amostra:

testeKSZoom <- ks.test(Zoom$Tempo, "pnorm", mean=mean(Zoom$Tempo), sd=sd(Zoom$Tempo))
testeKSZoom
## 
##  One-sample Kolmogorov-Smirnov test
## 
## data:  Zoom$Tempo
## D = 0.20017, p-value = 0.3517
## alternative hypothesis: two-sided

A partir das informaes contidas no teste:

A um nvel de significncia de 0.1 a hiptese nula no pode ser rejeitada chegando-se a concluso que a amostra segue uma distribuio normal.

A visualizao da distribuio cumulativa comparada com a distribuio cumulativa da curva normal mostrada em seguida, onde os dois pontos conectados entre as curvas demonstram a estatstica D do teste de Komogorov.

dist.Normal.Zoom<- rnorm(10000, mean(Zoom$Tempo), sd(Zoom$Tempo))

gera_ksplot(Zoom, dist.Normal.Zoom)

6 Teste de Levene

O teste de Levene utilizado para avaliar se a varincia entre os grupos homognea. O teste calcula a estatstica F como mostrado na frmula a seguir: \[ F = \frac{(N-k)} {(k-1)} \frac{\sum_{i=1}^{k}N_{i}(\bar{Z}_{i.}-\bar{Z}_{..})^{2} } {\sum_{i=1}^{k}\sum_{j=1}^{N_i}(Z_{ij}-\bar{Z}_{i.})^{2} } \] Entretanto, ao invs de calcular o quadrado da distncia de cada ponto para a mdia de sua amostra, o teste de Levene transforma cada ponto da amostra para o mdulo da diferena entre o ponto e a mdia ou mediana da amostra, como na frmula a seguir:

\[{\displaystyle Z_{ij}={\begin{cases}|Y_{ij}-{\bar {Y}}_{i\cdot }|,&{\bar {Y}}_{i\cdot }{\text{ a mdia do }}i{\text{-simo grupo}},\\|Y_{ij}-{\tilde {Y}}_{i\cdot }|,&{\tilde {Y}}_{i\cdot }{\text{ a mediana do }}i{\text{-simo grupo}}.\end{cases}}}\] A escolha entre utilizar a mdia ou a mediana est relacionada com a normalidade da amostra. Para amostras que seguem uma distribuio normal utilizada a mdia. Para amostras assimtricas o parmetro indicado a mediana.

O teste com centro na mediana tambm chamado de teste de Brown-Forsythe

O teste possui como hiptese nula a seguinte afirmao:
H0: As amostras foram retiradas de popules com disperses homogneas
